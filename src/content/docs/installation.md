# Installing NueScript

## System Requirements

- Node.js 18 or later
- npm 7 or later

## Quick Install

```bash
npm install -g nuescript
```

## Verifying Installation

```bash
nue --version
```

## Editor Setup

We recommend using Visual Studio Code with our official extension:

1. Install VS Code
2. Install the NueScript extension
3. Reload VS Code
